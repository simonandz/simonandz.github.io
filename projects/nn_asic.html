Neural Network ASIC Accelerator

Objective

To design a fixed-point neural-network accelerator (Q8.24) that performs matrix multiplication and activation functions, all within a single ASIC fabric.

Outcome & Contribution

• Architected a datapath supporting multiply‐accumulate operations for vector‐matrix inference.
• Implemented multiple activation blocks (ReLU, Sigmoid) in SystemVerilog.
• Synthesized the design using Yosys and OpenLane; achieved timing closure at 1 GHz.
• Validated functional correctness via post‐layout simulations.

Skills & Tools

  • SystemVerilog, Verilog (RTL design, testbenches)
  • Machine Learning (fixed-point quantization, Q8.24)
  • ASIC Design Flow (Yosys, OpenLane, timing analysis)
  • SPICE & Timing Verification
