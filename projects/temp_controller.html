On/Off Temperature Controller

Objective

Implement a relay-based on/off control for a heating pad using a PID feedback loop in Simulink, coupled with real-time thermometer data.

Outcome & Contribution

• Developed a Simulink model featuring a PID block controlling a digital relay.
• Tuned PID gains to maintain a target temperature within ±0.5 °C.
• Built a small test rig with an actual heating pad and microcontroller to validate simulation results.
• Documented temperature response curves and explained controller stability margins.

Skills & Tools

  • Simulink & MATLAB (PID design, block diagrams)
  • Feedback Control Systems (tuning, stability analysis)
  • Embedded Prototyping (microcontroller integration, relay wiring)
  • Data Acquisition (real-time temperature logging)
