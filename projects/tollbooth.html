Toll Booth Controller

Objective

Implement a finite-state machine for a toll booth on an FPGA using Multisim for SPICE modeling, then deploy on real hardware to track car passage and toll collection logic.

Outcome & Contribution

• Planned state transitions and combinational logic for “vehicle detected,” “gate open,” “payment validated,” and “gate close.”
• Built SPICE model in Multisim to verify timing and glitch-free operation.
• Synthesized the design, loaded onto an FPGA dev board, and assembled a physical toll-booth mockup to demonstrate live operation.
• Documented design flow in a technical report highlighting timing margins and logic utilization.

Skills & Tools

  • Multisim (SPICE circuit modeling)
  • Digital Design (FSM, Verilog for FPGA)
  • FPGA Deployment (board bring-up, timing constraints)
  • Technical Reporting (design documentation, timing analysis)
